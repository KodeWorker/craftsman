# 2026-4-17

## Basic Chat Interaction

Wired up end-to-end chat: FastAPI server streams completions over ndjson,
client retries until server is healthy then enters a conversation loop.
Reasoning tokens from llama.cpp (`reasoning_content` field) are displayed
in dim style before the assistant response. Token usage is forwarded in a
final `meta` chunk and shown in the banner alongside model and context info.
